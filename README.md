# 🎭 System Prank Simulator (C++)

Welcome to the **System Prank Simulator**! This is an educational project developed in **C++** to demonstrate the power of the Windows API and GDI (Graphics Device Interface) for creating visual effects and system-level simulations.

## 🚀 Features
* **Screen Melting Effect:** A cool visual distortion that makes the desktop look like it's melting.
* **Custom System Prompts:** Simulates realistic-looking Windows error messages and prompts.
* **Low-Level Interaction:** Demonstrates how to interact with the Windows Registry and system processes.

## 🛠️ How to Use
1.  Download the `.cpp` source code from this repository.
2.  Compile the code using a C++ compiler (e.g., MinGW, Visual Studio, or Dev-C++).
3.  Run the generated `.exe` file on your PC to see the effects.
